## Getting Started

### Quick Installation

Biograph is easy to set up on any Linux server. Here's how:

```bash
# Install Frappe Bench
pip install frappe-bench

# Initialize a new bench
bench init frappe-bench
cd frappe-bench

# Get ERPNext (required for full functionality)
bench get-app erpnext

# Get Biograph
bench get-app https://github.com/Tacten/biograph

# Create a new site
bench new-site yourhospital.com

# Install the apps
bench --site yourhospital.com install-app erpnext
bench --site yourhospital.com install-app healthcare

# Start the server
bench start
```

### Production Deployment

```bash
# Enable production mode
sudo bench setup production [your-user]

# Enable HTTPS
sudo bench setup lets-encrypt yourhospital.com

# Done! Your production server is ready
```

### Docker Deployment

```bash
# Use official Docker images
docker pull frappe/erpnext:latest

# Or use Docker Compose
git clone https://github.com/frappe/frappe_docker
cd frappe_docker
docker-compose up -d
```

## Frequently Asked Questions

<div class="faq-section">

### General Questions

**Is Biograph really free?**
Yes! Biograph, ERPNext, and Frappe Framework are all open source (GPL-3.0 license). There are no license fees, no per-user charges, and no feature limitations. You only pay for hosting, support (optional), and implementation services.

**What's the catch?**
There isn't one. Open source software is a proven model used by the world's largest companies. You get free software, and vendors like Tacten provide commercial support and services.

**How mature is Biograph?**
Biograph is built on Frappe Framework (since 2008) and ERPNext (since 2010), which are used by 10,000+ organizations worldwide. The healthcare modules have been in development since 2017.

**Can I use it for my hospital?**
Yes! Biograph is production-ready and being used by healthcare organizations worldwide, from small clinics to large hospital chains.

### Technical Questions

**What if I need a feature that doesn't exist?**
You have three options: 1) Build it yourself (low-code framework), 2) Hire a developer (large community), or 3) Request it from the community (many features are built this way).

**Can I integrate with my existing systems?**
Yes! Biograph has REST APIs, webhooks, and supports standard healthcare protocols (HL7, FHIR, DICOM). Integration with existing systems is straightforward.

**Is it HIPAA compliant?**
Biograph provides the tools for HIPAA compliance (encryption, audit logs, access controls, etc.), but compliance also depends on how you configure and host the system. Tacten offers HIPAA-compliant hosting.

**Can I customize it?**
Absolutely! That's the main advantage. You have complete access to source code and a low-code framework for most customizations. No vendor approval needed.

**What about mobile access?**
Biograph has a responsive design that works on any device. Native mobile apps are also available from the community.

### Business Questions

**What's the total cost of ownership?**
For a 100-user hospital: approximately $40K first year (including implementation), then $30K/year ongoing (hosting + support). Compare this to $470K+ for traditional systems.

**How long does implementation take?**
Depends on size and complexity: Small clinic (1-2 months), Medium hospital (3-6 months), Large hospital (6-12 months).

**Do you provide training?**
Yes! Tacten provides implementation services including training. There's also extensive documentation and community resources.

**What if we need support?**
Multiple options: Free community support, paid partner support, or Tacten's premium 24/7 support for critical deployments.

**Can we migrate from our current system?**
Yes! Data migration from most systems is possible. We recommend a phased approach to minimize disruption.

### Comparison Questions

**How does it compare to Epic/Cerner/Allscripts?**
Biograph offers similar clinical features at a fraction of the cost, with the added benefit of complete ERP integration and customization freedom. You own your data and system.

**Why choose Biograph over other open-source HIMS?**
Unique advantages: 1) Tight ERPNext integration (full ERP), 2) Low-code framework (easy customization), 3) Large community (50,000+ members), 4) Modern tech stack.

**What about OpenMRS or Bahmni?**
Those are excellent for public health and resource-limited settings. Biograph is better for hospitals needing full business operations (billing, inventory, HR, etc.) integrated with clinical workflows.
